AKKA ROAD WEBSITE STARTER

What this is:
- A simple one-page static website for Akka Road.
- Files included: index.html, styles.css, script.js, and SVG artwork.
- You can host it on Netlify, Vercel, GitHub Pages, Squarespace custom code, or any regular web host.

Before launch, edit:
1. booking@example.com in index.html
2. Social links in index.html
3. Music streaming links in index.html
4. Show dates in index.html
5. Replace placeholder media boxes with real photos/videos

How to link your domain:
1. Upload this site to a host.
2. In your hosting dashboard, add your custom domain.
3. In your domain registrar DNS settings, point the domain to your host using the host's DNS records.
   - Usually this means changing nameservers OR adding A/CNAME records.
4. Turn on HTTPS/SSL in the host dashboard.

Good easy hosts:
- Netlify: easiest for static sites; drag and drop the folder.
- Vercel: also easy.
- Squarespace/Wix: easier editor, but less custom.
- Shopify: only if selling merch is the main purpose.
